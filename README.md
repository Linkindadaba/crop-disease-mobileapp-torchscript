# Crop Disease Detector (Flutter + TorchScript)

A mobile app built with Flutter that detects crop diseases using a TorchScript MobileNetV2 model.

## Features
- Pick images from gallery or camera
- Run TorchScript model on-device
- Show predicted disease class

## Setup
1. Install Flutter SDK
2. Clone this repo
3. Run `flutter pub get`
4. Connect a device or emulator
5. Run `flutter run`

## Assets
- TorchScript model: `assets/mobilenetv2_plant_disease.pt`
